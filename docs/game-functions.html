Listening for click event

Next we're going to add some code so the board will click for user click events. 


    window.bind('%ltButton-1>', handle_click)

Then define a function named handle_click. In the body of the function add "I've been clicked". Now when you run the game and click on the board, it should read "I've been clicked" in the console.

Pass the event to the function, so we'll be able to grab the x,y coordinates of the click.


def handle_click(e):
    print("someone clicked", e.x, e.y)
