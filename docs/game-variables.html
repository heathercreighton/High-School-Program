Game Variables

Create a new Python Repl.it called `dots-and-lines-game` and save it to a new repo on your github org. You'll now have one repl.it for your labs, and one for your dots-and-lines game.

Add the following to your `main.py` file:


    size_of_board = 600
    number_of_dots = 6
    symbol_size = (size_of_board / 3 - size_of_board / 8) / 2
    symbol_thickness = 50
    dot_color = '#7BC043'
    player1_color = '#0492CF'
    player1_color_light = '#67B0CF'
    player2_color = '#EE4035'
    player2_color_light = '#EE7E77'
    Green_color = '#7BC043'

If you'd like to choose different hex colors, this color picker may be helpful to you.

Add the following variables:

  • `distance_between_dots` set equal to the size of the board divided by the number of dots.
  • `dot_width` set equal to 25% times the size of the board divided by the number of dots.
  • `edge_width` set equal to 10% times the size of the board divided by the number of dots.
